# JSRay Brand Assets

These files are first-pass SVG logo drafts for JSRay.

## Direction

The current direction uses angle brackets around a small ray point:

- `jsray-mark.svg`: square mark for README, social previews, docs, and app surfaces.
- `jsray-logo.svg`: horizontal lockup with wordmark and descriptor.
- `jsray-favicon.svg`: simplified icon for small sizes.

The mark is intentionally simple: it should read as code at favicon size while keeping the "ray" idea visible without decorative complexity.

## Palette

- Ink: `#202124`, `#31404A`
- Ray mint: `#5DD8B0`
- Ray amber: `#FFCB6B`
- Ray cyan: `#5AC8FA`
- Surface: `#F8FAFC`
- Border: `#E2E8F0`

## Notes

This is a design route, not a final trademark-cleared identity. Before public release, test the mark at 16, 24, 32, 64, and 128 px, then outline the wordmark if the typography direction is selected.
