# JSRay Logo — 11b (dark-base spectrum `</>`)

Colors: cyan `#22D3EE` → indigo `#818CF8` → magenta `#E879F9` · dark base `#0E1018`

## Files

- `jsray-mark-*.svg / .png` — standalone icon (app icon / favicon), 160×160 rounded square
- `jsray-logo-*.svg / .png` — icon + JSRay wordmark, horizontal lockup
- `-dark` = dark-background version (dark surfaces / dark UI)
- `-light` = light-background version (light surfaces / print)

SVG files are the preferred vector sources; PNGs are high-resolution bitmaps, ready to use as-is.
The wordmark is set in Helvetica Neue / Helvetica / Arial.
